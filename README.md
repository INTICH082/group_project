
# 🎯 EduTest

> Система для массового тестирования и опросов с веб-интерфейсом и Telegram-ботом

![Architecture](https://via.placeholder.com/800x400/4F46E5/FFFFFF?text=EduTest+System)

## ✨ Возможности

### 👤 Для студентов
- 📝 Доступ к вопросам курсов
- ✅ Проверка ответов в реальном времени
- 📊 Просмотр результатов

### 👨‍🏫 Для преподавателей
- ➕ Создание вопросов с вариантами ответов
- 📚 Управление курсами
- 👥 Просмотр результатов студентов

### 👑 Для администраторов
- 🔧 Управление ролями пользователей
- 👁️ Полный доступ к системе

## 📊 Схема базы данных

```sql
-- Основные таблицы:
users (id, role)
questions (id, text, options[], correct_answer, author_id)
courses (id, name, teacher_id)
course_questions (course_id, question_id)
results (user_id, question_id, user_answer, is_correct)
```


## 🎮 Демо-сценарии

### 🎓 Студент
1. Получить вопросы курса: `GET /course/1/questions`
2. Ответить на вопрос: `POST /answer`
3. Получить результат проверки

### 👨‍🏫 Преподаватель  
1. Создать вопрос: `POST /question`
2. Добавить вопрос в курс: `POST /course/add-question/`
3. Просмотреть вопросы курса

### 👑 Админ
1. Изменить роль пользователя: `PUT /user/3/role`
2. Проверить доступ ко всем функциям
